SMS ve Bulut Santral API ile Uygulamalarınıza Güçlü İletişim

Kolay entegrasyon, yüksek güvenilirlik, 7/24 destek. SMS gönderimi ve bulut santral özelliklerinizi tek bir API'de birleştirin.

curl --request GET \
  --url 'https://sms.verimor.com.tr/v2/headers?username=KULLANICI_ADI&password=SIFRE' \
  --header 'accept: application/json'
import requests

url = 'https://sms.verimor.com.tr/v2/headers'
params = {
    'username': 'KULLANICI_ADI',
    'password': 'SIFRE'
}
headers = {
    'accept': 'application/json'
}

response = requests.get(url, params=params, headers=headers)
print(response.json())
const axios = require('axios');

const url = 'https://sms.verimor.com.tr/v2/headers';
const params = {
  username: 'KULLANICI_ADI',
  password: 'SIFRE'
};

const response = await axios.get(url, {
  params: params,
  headers: {
    'accept': 'application/json'
  }
});

console.log(response.data);
